# 变更日志

## 2026

- **Functions 重构**: 云函数数据模型简化，去除版本表和分支维度；Deno Functions Runtime 重写，使用 file:// 部署模式；前端新增 Supabase 风格多文件编辑器和模板系统
- **GraphQL 惰性加载**: 重构 GraphQL 管理器实现惰性加载和缓存清理
- **MCP Server**: 新增 15 个 MCP Tools（ProjectTools / ModelingTools / DataTools），基于 Quarkus MCP Server HTTP 扩展

## 2025

- **云函数 (Functions)**: 首次引入云函数能力，Deno Functions Runtime + Worker 隔离执行
- **对象存储 (Storage)**: 完整实现 S3 存储后端（AWS SDK v2），20 个测试全部通过
- **任务调度 (Job)**: 完善触发器配置和执行日志，与 Flow 模块联动

## 2024

- **流程编排 (Flow)**: 可视化流程设计器，BPMN 风格节点（开始/结束/排排网关/服务任务/用户任务/调用活动），插件体系
- **任务调度 (Job)**: 内置触发器与作业执行引擎，支持 Cron、延时及事件驱动
- **对象存储 (Storage)**: 统一抽象本地存储和 S3 存储
- **身份认证 (Auth)**: RBAC 权限体系，项目级 OIDC/LDAP/Script 身份提供商

## 2023

- **初始版本**: 核心引擎（flexmodel-engine），动态建模与多数据源支持
- **数据建模**: 支持 10+ 数据库的运行时动态建模
- **GraphQL**: 自动生成 GraphQL Schema 和查询接口
- **REST API**: 标准化 REST API 接口
