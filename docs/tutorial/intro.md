# 介绍

## 简介

Flexmodel 提供了运行时动态构建 API 服务的能力，帮助用户创建自定义服务接口。前端开发人员可以利用这些配置好的接口来开发自己的页面，从而提高开发效率。

### Flexmodel 具备哪些功能？

1. API服务：支持通过GraphQL构建RestAPI服务
2. 数据源管理：支持主流数据库和国产化数据库的接入
3. 数据建模：支持数据模型管理，数据的管理
4. 身份认证：RBAC权限体系、OIDC/LDAP/社交账号、JWT安全验证
5. 流程编排：可视化流程设计器，后端业务逻辑编排
6. 任务调度：触发器与作业执行引擎，支持 Cron、延时及事件驱动调度
7. 对象存储：统一抽象 S3、OSS 与本地存储，轻松管理 Bucket 与文件对象
8. AI工具集成：基于 MCP 协议开放核心工具，让 AI 客户端直接管理项目与数据
9. 云函数：Deno Sidecar 隔离执行的云函数能力，支持多文件源码编辑和模板系统（实验性）

### Flexmodel 能做什么？

* 统一数据服务，将一些通用的接口直接提供给前端，提高开发效率。

* BFF层或者API网关，将后端的接口转化为GraphQL请求或者RestAPI请求，返回给前端

* 低代码数据引擎，将数据模型转换为数据服务给到前端低代码页面
