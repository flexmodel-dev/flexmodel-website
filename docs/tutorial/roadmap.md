# 路线图

Flexmodel 正在持续演进，以下是我们的发展规划。路线图按阶段划分，每个阶段聚焦核心能力的提升与生态的完善。

:::tip 参与贡献
路线图中的部分功能仍在规划阶段，欢迎社区开发者参与讨论和贡献。你可以在 [GitHub](https://github.com/flexmodel-dev/flexmodel) 提交 Issue 或 PR。
:::

## ✅ 已完成

| 功能 | 说明 | 完成时间 |
|---|---|---|
| **动态建模与多数据源** | 运行时动态建模，支持 SQLite/MySQL/PostgreSQL/Oracle/DB2/MongoDB 等 10+ 数据库 | 2023 |
| **GraphQL 自动生成** | 根据数据模型自动生成 GraphQL Schema 和查询接口 | 2023 |
| **REST API** | 标准化 REST API 接口，数据 CRUD 一键生成 | 2023 |
| **身份认证 (Auth)** | RBAC 权限体系、项目级 OIDC/LDAP/Script 身份提供商、JWT 安全验证 | 2024 |
| **流程编排 (Flow)** | 可视化流程设计器，BPMN 风格节点，插件体系，后端业务逻辑编排 | 2024 |
| **任务调度 (Job)** | 内置触发器与作业执行引擎，支持 Cron、延时及事件驱动调度 | 2024 |
| **对象存储 (Storage)** | 统一抽象本地/S3/OSS 存储，完整 S3 实现（AWS SDK v2） | 2024–2025 |
| **MCP Server** | 15 个 MCP Tools（Project/Modeling/Data），基于 Quarkus MCP Server HTTP 扩展 | 2025 |
| **云函数 (Functions)** | Deno Functions Runtime + Worker 隔离执行，多文件源码编辑和模板系统 | 2025–2026 |

## 📋 近期规划（2026 Q3–Q4）

### 非结构化数据支持

- **支持向量字段**：向量字段支持向量索引，支持向量搜索、向量聚合、向量距离计算等操作
- **向量搜索融入现有查询体系**：不是独立世界——用户做"结构化过滤 + 语义排序"是一个请求
- **文件向量存储**：支持文件的向量索引，向量搜索等操作

### 开发体验

- **CLI 工具**：命令行管理项目、模型、数据，支持脚本化部署和 CI/CD 集成
- **SDK 完善**：TypeScript/Python/Java SDK，覆盖建模、数据、认证等核心 API
- **本地开发模式增强**：`flexmodel dev` 一键启动开发环境，热重载、Mock 数据生成

### 安全与合规

- **审计日志**：操作审计记录，支持查询和导出
- **数据加密**：字段级加密存储，传输层 TLS 强化
- **合规报告**：数据访问权限报告生成
